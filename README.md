# 🌿 Wellness / Mood Tracker App

### Usability Evaluation Web Report

## 📌 Project Overview

This project is a **static web-based documentation platform** that presents the usability evaluation of a **Wellness / Mood Tracker Application**.

The website showcases the results of two key Human-Computer Interaction (HCI) evaluation methods:

* **Heuristic Evaluation** based on *Jakob Nielsen’s 10 Usability Heuristics*
* **Usability Testing** conducted with actual respondents

It serves as a **central repository of findings, issues, and design recommendations** to improve the user experience of the application.

---

## 🎯 Objectives

* Apply usability evaluation methods to a developed prototype
* Document evaluation processes in a structured web format
* Analyze usability issues using established HCI principles
* Present findings and recommendations clearly
* Demonstrate front-end development skills using HTML, CSS, and JavaScript

---

## 🧪 Evaluation Methods

### 🔍 Heuristic Evaluation

Each group member conducted an individual evaluation using:

* Jakob Nielsen’s 10 Usability Heuristics
* A structured Heuristic Evaluation Workbook

### 👥 Usability Testing

The group conducted usability testing with selected respondents to:

* Observe user interaction with the prototype
* Identify usability issues
* Gather feedback through surveys and task-based testing
