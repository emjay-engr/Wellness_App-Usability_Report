📌 Project Overview

About the Wellness Super App

A personalized self-care mobile application designed to help users track daily habits, plan tasks, monitor mood, and view analytics — all in one place.

📱

Prototype Type

High-fidelity mobile prototype in Figma for iPhone 14 Plus, featuring 8 interconnected screens.

🎯

Target Users

BSIT students and young professionals aged 18–25 managing wellness, habits, and daily tasks.

🔍

Evaluation Methods

Heuristic Evaluation using Nielsen's 10 Heuristics + Usability Testing with real respondents.

📅

Evaluation Period

March 2026 · Partido State University, Goa, Camarines Sur
