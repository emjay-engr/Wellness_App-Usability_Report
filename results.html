🔴 Issues Identified

Key Usability Problems

Issues ranked by severity based on heuristic evaluation findings from all 5 group members.

🔴
Critical · H9

No Error Validation Messages

All 5 evaluators noted that submitting forms with empty required fields gives no feedback. Users cannot tell if an action worked or failed — identified as the most critical issue overall.

🔴
Critical · H10

No Onboarding or Help Documentation

First-time users do not understand the Intensity Slider or streak dot color system. No tutorial, tooltip, or help screen exists to guide new users through the app's features.

🟡
Moderate · H3

No Undo for Deleted Items

When a task or habit is deleted, there is no confirmation dialog or undo option. This was reported by 3 out of 5 respondents as a concern during usability testing.

🟡
Moderate · H9

Missing Empty State Screens

Analytics and past entries show blank space when no data is available. There is no helpful "No data yet" message or call-to-action to guide users forward.

🟢
Minor · H7

No Swipe Gestures for Power Users

Frequent users cannot swipe to mark habits done or dismiss tasks. Only tap-based interactions are available, which reduces efficiency for experienced users.

💬 Discussion

Discussion of Findings

Interpretation and analysis of the usability evaluation results.

✅

Strong Overall Performance

The Wellness Super App received positive usability scores across navigation, visual design, clarity, and utility — averaging 4.2 out of 5.0. Respondents found the app visually appealing and easy to understand thanks to consistent green color system, emoji-paired labels, and clean card-based layouts.

⚠️

Lowest Score: Error-Free Experience

The lowest score (3.4/5.0) was recorded for the "Error-Free Experience" category. This aligns with the two critical heuristic violations — absence of form validation (H9) and lack of onboarding help (H10). These are the highest-priority improvements for the next prototype iteration.

💚

High User Recommendation Rate

Despite the gaps found, 84% of respondents said they would recommend the app to others, and all 5 agreed it would genuinely help manage their wellness routine. This confirms the core concept and design direction are solid — improvements are refinements, not redesigns.

✅ Recommendations

Design Improvement Recommendations

Actionable fixes for each identified usability issue, prioritized by severity.

🛡️
Critical Fix

Add Form Validation Messages

Show inline error messages when required fields are empty. Use red border highlight with helper text: "Please select your mood first before saving."

📖
Critical Fix

Add Onboarding Tutorial

Create a 3-step onboarding carousel on first launch that explains the Mood Tracker intensity slider, streak dot colors, and how to create a habit.

↩️
Moderate Fix

Add Undo / Confirm Delete

Show a confirmation bottom sheet before deleting a habit or task. Add a 3-second "Undo" toast notification after deletion for user safety.

📭
Moderate Fix

Design Empty State Screens

Replace blank analytics charts and entry lists with illustrated empty state screens that show a friendly message and encourage users to start logging data.

👆
Minor Enhancement

Add Swipe Gestures

Allow users to swipe right to mark a habit done and swipe left to delete or reschedule a task — following common mobile UX conventions for efficiency.
